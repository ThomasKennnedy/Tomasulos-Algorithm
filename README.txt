------------Build Instructions------------
The project was developed on the CS665 Linux Virtual Machine.
The project was run under an X-Windows session using X-Win 32.

To build the project using the included makefile run:
     make clean
     make jar
     
To run the project:
     java -jar ujwal-kennedy.jar
